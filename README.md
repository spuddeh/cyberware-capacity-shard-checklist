# Cyberware Capacity Shard (and Cyberjunkies) Checklist

**Cyberware Capacity Shard Checklist** tracks all of the Phantom Liberty Cyberware Capacity Shards and Cyberjunkies in Dogtown so you never have to alt-tab to a wiki.

📥 **Download:** [Cyberware Capacity Shard (and Cyberjunkies) Checklist on Nexus Mods](https://www.nexusmods.com/cyberpunk2077/mods/27806)

Phantom Liberty hides 6 Cyberware Capacity Shards across Dogtown (dropped by specific Cyberjunkie NPCs and one cache during "Dog Eat Dog"). There are also 6 additional Cyberjunkies who do not drop a shard but are part of the same quest line. This mod tracks all 12.

## 🚀 Features

* **In-Game Checklist:** All 12 entries across 2 categories (6 Cyberware Capacity Shards + 6 shardless Cyberjunkies).
* **Quest-Fact Detection:** Defeated Cyberjunkies are detected automatically using in-game quest facts. Shard Cyberjunkies also verify the shard was looted, so a kill alone will not falsely complete a shard entry.
* **Spawn Requirements:** The prerequisite for each gated Cyberjunkie is shown, and it will not flag one as detectable before it can actually spawn.
* **Reactive Proximity Scanner (v2.0.0):** Built on 0-Engine. Reacts the moment you cross into range. No polling timer, so no CPU cost when you are away.
* **Following Markers:** Cyberjunkie detection markers follow the NPC instead of sitting on a fixed point (the Stadium cache uses a static marker).
* **Smart Pause:** Suppressed during loading screens, fast travel, and menus.
* **Set Pin & Lazy Mode:** Standalone waypoint button; optional Lazy Mode adds Teleport and Unstuck.
* **Per-Character Persistence:** Progress is tied to each save file.

## 📋 Requirements

* **Phantom Liberty** expansion.
* [Cyber Engine Tweaks](https://www.nexusmods.com/cyberpunk2077/mods/107).
* [0-Engine](https://www.nexusmods.com/cyberpunk2077/mods/27967) (pure CET-only build). New required dependency as of 2.0.0; it has its own requirements listed on its mod page.

## 🎮 Usage

1. Open the CET overlay.
2. Open the **Cyberware Capacity Shard Checklist** window.
3. Click the checkboxes to mark progress manually, or let automatic tracking handle it.
4. Click **Set Pin** to drop a waypoint at an entry's location.
5. Adjust the detection range in Settings (default 50m, adjustable 25m to 100m).

## ⚠️ Notes

* The **Stadium Parking Cache** is inside a van during the "Dog Eat Dog" mission and cannot be accessed afterwards. If you miss it, use Lazy Mode. You cannot open the van doors, but you can interact with the container by aiming at the door handle in the right spot.
